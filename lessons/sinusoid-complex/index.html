電気回路Ⅱ　第8回

正弦波の表現

過渡現象（前半）を終え、ここから交流回路へ。正弦波を複素数で表し、面倒な微分方程式を“掛け算・割り算”に変える道具を身につける回です。

この回の到達目標
  • 正弦波を A cos(ωt+φ) の形で読み書きできる。
  • 複素数 z=x+jy を複素平面の矢印として捉え、四則演算ができる。
  • オイラーの公式 e±jθ=cosθ±j sinθ と極形式を説明できる。
  • 「複素数を掛ける＝回転（位相をずらす）」を理解する。

8.1　直流と交流

変わらない電気と、振動する電気

上：直流（一定）／下：交流（正弦波状に振動）

直流 (DC)：電圧・電流が時間によらず一定。乾電池などが代表例。
交流 (AC)：時間とともに規則正しく振動する。家庭のコンセントの電気がこれ。

交流電圧の一般形：
v(t) = V0 cos(ωt + φ)

後半（第8回〜）では、この交流を扱う「交流回路」を学びます。まずは交流の正体＝正弦波を詳しく見ていきましょう。

解説図

8.2　正弦波交流

正弦波 sin と 余弦波 cos

sin x と cos x

どちらも山と谷をなめらかに繰り返す、同じ形の波。値は +1 〜 −1 の範囲を行き来します。

  • sin x は x=0 で 0 から始まる。
  • cos x は x=0 で 1 から始まる。
  • 2つは横方向にずれて並ぶ（位相のずれ）。cos x = sin(x+90°)。
  • 一定間隔で同じ波形を繰り返す＝周期的な波。

正弦波交流回路（抵抗のみ）

オームの法則そのままで解ける

抵抗だけの回路なので、直流と同じくオームの法則 v = R i が使えます。

オームの法則
v(t) = R · i(t)
左辺に正弦波電圧を代入
V0 cos(ωt+φ) = R · i(t)
両辺を R で割る
i(t) = (V0/R) cos(ωt+φ)

電流の振幅は V0/R。位相 φ は電圧と同じ＝電圧と電流は同相。直流と同じく、一回の計算で求まります。

8.3　複素数（複素ベクトル）

複素数は、平面上の矢印

複素平面：横＝実軸 Re、縦＝虚軸 Im

横方向の軸を実軸（Re）、縦方向の軸を虚軸（Im）とし、この2本でできる平面を複素平面と呼びます。

複素数 z は、原点から伸びる矢印（複素ベクトル）。先端の横成分が x、縦成分が y です。

解説操作

複素数の実部と虚部

z = x + jy

z = x + j y
実部（real part）　　虚部（imaginary part）

x を実部（実軸の値）、虚数単位 j のついた y を虚部（虚軸の値）と呼びます。 j は虚数単位で j² = −1。数学の i は電流と紛らわしいので、電気回路では j を使います。

操作：矢印の先をドラッグ
解説操作

共役複素数（きょうやくふくそすう）

共役（きょうやく）は、実軸で折り返した鏡像

z = x + jy　に対して　z̄ = x − jy

実部 x はそのまま、虚部の符号だけを反転させた複素数を共役複素数（きょうやく）（記号 z̄）と呼びます。複素平面では実軸に関して対称な点＝鏡像です。後で割り算に効いてきます。

操作：z をドラッグして z̄ の動きを見る

z　 z̄

解説操作

偏角と大きさ

矢印を「長さ」と「向き」で測る

大きさ |z| = √(x² + y²)　　偏角 θ = arctan(y/x)

大きさは矢印の長さ（三平方の定理）、偏角は矢印が実軸となす角度。この2つで複素数を表すのが、後で出る極形式です。

操作：z をドラッグ（円弧＝偏角、矢印長＝大きさ）
解説図

特殊な場合

実軸上・虚軸上の複素数

z₁=2（実数）／z₂=j3（純虚数）

虚部が0のとき：実軸上の点＝ふつうの実数。例 z₁=2。矢印は実軸と同じ向きなので偏角 0°（0 rad）。

実部が0のとき：虚軸上の点＝純虚数。例 z₂=j3。矢印は真上を向くので偏角 90°（π/2 rad）。

足し算は、矢印の継ぎ足し

z1+z2 = (x1+x2) + j(y1+y2)　実部どうし・虚部どうしを足す

作図：① z1 を描き、② z2 を z1 の先端から継ぎ足し、③ 原点から継ぎ足した先端まで結ぶ。これが和（減算は実部・虚部を引くだけ）。

複素数の加算・減算（数値例）

具体的な数で確かめる

z1 = 3 + j5、　z2 = 1 − j2
和
z1+z2 = (3+1) + j(5−2) = 4 + j3
差
z1−z2 = (3−1) + j(5+2) = 2 + j7

ポイントは「実部どうし・虚部どうし」で足し引きすること。引き算では −(−2)=+2 の符号に注意。

解説

複素数の乗算

掛け算は展開して j²=−1

z1z2 = (x1+jy1)(x2+jy2)
多項式と同じく展開
= x1x2 + jx1y2 + jx2y1 + j²y1y2
j² = −1 を使う
= (x1x2 − y1y2) + j(x1y2 + x2y1)
数値例 z1=3+j5, z2=1−j2 のとき
= (3·1 − 5·(−2)) + j(3·(−2) + 1·5) = (3+10) + j(−6+5) = 13 − j1

オイラーの公式

指数関数と三角関数を結ぶ

e±jθ = cos θ ± j sin θ

指数関数 e の累乗が、三角関数 cos・sin に、虚数単位 j を通じて結びつきます。複号（±）は、左辺の肩の符号と右辺 sin の前の符号が対応します（上が上、下が下）。

この公式が、交流回路を複素数で扱うための要。次の2ページで導出します。

解説

オイラーの公式の導出 ①

準備：ネイピア数とマクローリン展開

ネイピア数 e（微分しても形が変わらない）
d/dx ex = ex　／　e = limn→∞(1+1/n)n ≈ 2.718
マクローリン展開（関数を多項式で表す）
f(x) = f(0) + f′(0)x + f″(0)/2! x² + f‴(0)/3! x³ + …

係数は「もとの関数を何回か微分して x=0 を代入」して決まります。次ページで ejx に当てはめます。

解説図

オイラーの公式の導出 ②

偶数次＝cos、奇数次＝j sin

ejx = 1 + jx + (jx)²/2! + (jx)³/3! + (jx)⁴/4! + …
j²=−1, j³=−j, j⁴=1 … を使って整理
= [1 − x²/2! + x⁴/4! − …] + j[x − x³/3! + x⁵/5! − …]
青 = cos x のマクローリン展開
赤 = j × sin x のマクローリン展開
∴ ejx = cos x + j sin x

極形式の乗算・除算

大きさは掛け算/割り算、偏角は足し算/引き算

z1=r1ejθ₁, z2=r2ejθ₂ のとき（指数法則 eaeb=ea+b）
乗算：z1z2 = r1r2 · ej(θ₁+θ₂)
除算：z1/z2 = (r1/r2) · ej(θ₁−θ₂)

直交形式の割り算は共役をかける手間がありますが、極形式なら「大きさを割り、角度を引く」だけ。次ページで掛け算の図形的な意味＝回転を見ます。

まとめ

今日のゴール

  • 正弦波は A, ω, φ の3つで決まる。抵抗だけなら電圧と電流は同相。コイルが入ると位相が遅れ、微分方程式になる。
  • 複素数 z = x + jy は複素平面の矢印。共役（きょうやく）・偏角・大きさで特徴づけられる。
  • 四則演算：和差は成分ごと、積商は j²=−1 と共役がコツ。
  • オイラーの公式 e±jθ=cosθ±j sinθ で、複素数は r·ejθ の極形式に。
  • 極形式の積は「大きさ掛け算・偏角足し算」＝回転。これが交流の位相を扱う土台。
