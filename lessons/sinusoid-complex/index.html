正弦波の表現

過渡現象（前半）を終え、ここから交流回路へ。正弦波を複素数で表し、面倒な微分方程式を“掛け算・割り算”に変える道具を身につける回です。

この回の到達目標
  • 正弦波を A sin(ωt+φ) の形で読み書きできる。
  • 複素数を複素平面の矢印として捉え、四則演算ができる。
  • オイラーの公式 ejθ=cosθ+j sinθ と極形式を説明できる。
  • 「複素数を掛ける＝回転（位相をずらす）」を理解する。

8.1　直流と交流

変わらない電気と、振動する電気

電池のような直流（DC）は時間によらず一定。家庭のコンセントの交流（AC）は規則正しく振動します。切り替えて見比べましょう。

交流は v(t) = V0 cos(ωt) のように、振幅・速さ（角振動数）・位相で形が決まります。次のスライドでそれを直接いじってみます。

正弦波交流回路（抵抗のみ）

抵抗だけなら、オームの法則そのまま

v(t) = V0 cos(ωt) を抵抗 R にかけると、電流は i = v/R。電圧と電流は同じタイミング（同相）で振動します。

正弦波交流回路（LR回路）

コイルが入ると、位相がずれる

抵抗とコイルの直列回路では R i + L di/dt = V0 cosωt。微分方程式です。コイル L を増やすと、電流は小さく・遅れていきます。

毎回この微分方程式を解くのは大変……これを複素数の掛け算・割り算で一気に出せるようにするのが、この先の目標です。

8.3　複素数（複素ベクトル）

複素数は、平面上の矢印

複素数 z = x + jy は、横（実軸 Re）に x、縦（虚軸 Im）に y 進んだ点への矢印。矢印の先をドラッグしてみてください。（j は虚数単位、j² = −1）

数学では虚数単位に i を使いますが、電気回路では電流 i と紛らわしいので j を使うのが慣習です。

複素数の実部・虚部・共役

共役は、実軸で折り返した鏡像

z = x + jy の共役複素数 z̄ = x − jy は、虚部の符号だけ反転。複素平面では実軸に関して対称な点です。z をドラッグして確かめましょう。

z（元の複素数）／ z̄（共役）。実部はそのまま、虚部の符号が反転します。

偏角と大きさ

矢印を「長さ」と「向き」で測る

大きさ |z| = √(x²+y²)（三平方の定理）、偏角 θ = arctan(y/x)。これが後で出る極形式の正体です。ドラッグしてみましょう。

緑の円弧が偏角 θ、矢印の長さが大きさ |z|。実軸上なら θ=0、虚軸上なら θ=90°(=π/2) になります。

足し算は、矢印の継ぎ足し

実部どうし・虚部どうしを足し引きするだけ。図では z2 を z1 の先に継ぎ足した先端が和になります。2本の矢印をドラッグしてみましょう。

オイラーの公式

ejθ = cos θ + j sin θ

この回の山場。大きさ1の複素数 ejθ は単位円の上の点で、その実部が cos θ、虚部が sin θ。円周上の点をドラッグして、影（射影）の動きを見てください。

直交形式と極形式

同じ1つの複素数、2通りの言い方

x + jy（直交形式）と r·ejθ（極形式）は同じ点。矢印をドラッグしても、下の r・θ を動かしても、両方の表示が連動します。

まとめ

今日のゴール

  • 正弦波は A, ω, φ の3つで決まる。抵抗だけなら電圧と電流は同相。
  • 複素数 z = x + jy は複素平面の矢印。共役・偏角・大きさで特徴づけられる。
  • 四則演算：和差は成分ごと、積商は j²=−1 と共役がコツ。
  • オイラーの公式 ejθ=cosθ+j sinθ で、複素数は r·ejθ の極形式に。
  • 極形式の積は「大きさ掛け算・角度足し算」＝回転。これが交流の位相を扱う土台。
